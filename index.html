Time Tracker

Time Tracker is a productivity-focused web app built with vanilla JavaScript and Bootstrap. It features a two-hour countdown timer, visual progress ring, pause tracking, and secure user authentication via Firebase. Designed with scalability in mind, it supports real-time data syncing and activity logging, laying the foundation for future features like performance dashboards and AI-generated revision flashcards.
